MB66 - Kiểm tra tài khoản nghi vấn theo đại lý

Chỉ kiểm tra tài khoản có Tổng nạp > 0, và chỉ đối chiếu các tài khoản trong cùng một đại lý. Sảnh cược chỉ hiển thị khi bấm Xem chi tiết.
File đại lý: A = hội viên, B = thời gian đăng ký, C = họ tên, D = đại lý cấp trên, F = tổng nạp, G = IP đăng nhập lần cuối, H = thời gian nạp cuối, I = ngân hàng, J = chi nhánh ngân hàng.
File cược game: C = đại lý cấp trên, D = hội viên, E = sảnh game, H = tổng cược hợp lệ của sảnh game đó.
Cột TK nghi vấn / Tổng TK hiển thị số tài khoản nghi vấn trong nhóm so với tổng tài khoản có nạp của đại lý. Đăng ký sớm nhất, nạp cuối gần nhất và khoảng cách ngày chỉ hiển thị trong phần Xem chi tiết.
